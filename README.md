Data Cleaning & Reporting Automation

Project Overview

This project demonstrates how to automate data cleaning and reporting using Python. The dataset contains duplicate records, missing values, and inconsistent text data. The project cleans the data and generates a visual sales report.

Objective

- Remove duplicate records
- Handle missing values
- Standardize inconsistent data
- Generate automated reports and visual summaries

Tools Used

- Python
- Google Colab
- Pandas
- Matplotlib

Dataset

The dataset contains:

- Name
- Age
- City
- Sales

Data Cleaning Tasks

1. Removed duplicate rows
2. Filled missing Age values using the mean
3. Filled missing Sales values using the mean
4. Standardized city names

Reporting

A bar chart was generated showing total sales by city.

Project Files

- sales_data.csv
- cleaned_data.csv
- sales_report.png
- Data_Cleaning_Project.ipynb

Output

- Cleaned dataset
- Automated visual report

Result

The project successfully automated the data cleaning process and generated a visual report, improving data quality and reporting efficiency.
